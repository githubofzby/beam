# 9. 当前复杂度概览

## Divide

近似：

```
O(D(n+m)+sorting)
```

其中：

- D = divide_hash_dims

------

## Prepare

原边扫描：

```
≈ O(n+m)
```

但：

- parallel/block/refinement 会产生额外重复。

------

## Proxy discovery

复杂度：

```
Σx C(bucket_size[x],2)
```

不能简单写：

```
C(group_size,2)
```

------

## Top-k

每节点：

```
proxy candidates sorting
```

------

## Exact scoring

```
Σ candidate (|adj[A]|+|adj[B]|)
```

------

## Matching

```
O(C log C)
```

------

## Update_S

```
O(|A|+|B|)
```

------

## Encoding

包含：

- 原图扫描
- sorting
- dense correction materialization
